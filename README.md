# test-code-review-integration

API đơn giản với **Node.js**, **TypeScript** và **Express**.

## Cài đặt

```bash
npm install
```

## Chạy

- **Development** (hot reload):

  ```bash
  npm run dev
  ```

- **Production** (build rồi chạy):

  ```bash
  npm run build
  npm start
  ```

## Endpoints

- `GET /` – Lời chào
- `GET /health` – Health check

Mặc định server chạy tại: http://localhost:3000
